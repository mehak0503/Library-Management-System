************ BUILD INSTRUCTIONS ****************
1. Using g++
> g++ main.cpp db.cpp Menu.cpp -lsqlite3
> ./a.out

2. Using IDE
simply run main.cpp with adding sqlite3 path to linker


*************** RUN SYSTEM *******************
1. Enter "admin" as password
2. Choose 1/2/3 for staff/user/supplier and 4 for exit.
3. Choose suitable option based on the operation required as presented in the menu.
4. Enter valid input wherever prompted.


************** TEST CASES ********************
1. Input for test cases is provided in testcases folder.
2. Makefile executes all the defined testcases.
3. Run "make" command to execute testcases.
